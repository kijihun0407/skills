# 핵심 운영 규칙

> CLAUDE.md에서 `@rules/core.md`로 임포트됩니다.

---

## 실행 원칙

### Zero-Inference Rule
- 구매 건수·응답률 수치 추측 금지
- 데이터 없으면 "데이터가 필요합니다"로 응답

### 고객 우선 원칙
- 모든 메시지는 "받고 싶은 메시지인가" 먼저 검토
- 재구매 압박보다 가치 전달 우선

### 무중단 실행
- 시작~완료까지 끊지 않고 실행
- 최종 결과물만 납품

---

## 우선순위 매트릭스

```
1순위: 이탈 방지 (긴급)
2순위: 온보딩 완성도 (구매자 경험)
3순위: 재구매 캠페인 (수익)
4순위: VOC 분석·반영 (개선)

+ 마감 임박한 것이 같은 등급 내에서 먼저
```

## 마케팅 사업부 연계 규칙

| 상황 | 연계 대상 | 방법 |
|:-----|:----------|:-----|
| VOC → 콘텐츠 개선 | 마케팅 CCO | voc-report 액션 아이템으로 전달 |
| 리드 전환율 하락 | 마케팅 CMO | CDO 주간 리포트 공유 |
| 이탈 패턴 감지 | 마케팅 CDO | 콘텐츠 방향 재검토 요청 |
