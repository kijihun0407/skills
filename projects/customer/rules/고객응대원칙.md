# 고객 응대 원칙

> 모든 고객관리 작업 전 자동 체크합니다.
> 하나라도 해당되면 작업 중단 후 사용자에게 알립니다.

---

## 절대 금지

1. **구매 직후 즉시 재구매 유도**
   - D+3 이전에 상품 구매 제안 금지
   - 온보딩 메시지에 가격·상품 링크 삽입 금지

2. **가격 인하·희소성 조작 표현**
   - "할인", "세일", "오늘만", "선착순" 류 금지
   - 사실에 근거한 출시 안내는 허용

3. **성과 보장 표현**
   - "이걸 쓰면 반드시", "100% 효과" 류 금지
   - 구매자 후기는 사실 그대로만 인용

4. **개인정보 외부 노출**
   - DM 내용·구매 정보 외부 언급·공유 금지

---

## 작업 전 자동 체크

```
[ ] 1. D+3 이전 재구매 언급 없음?
[ ] 2. 가격 인하·희소성 조작 표현 없음?
[ ] 3. 성과 보장·과장 표현 없음?
[ ] 4. 개인정보 보호 원칙 준수?
```

**4개 통과** → 진행
**1개라도 해당** → 작업 중단:
> "⚠️ 고객응대 원칙 위반: [항목 번호] [내용]. 수정 후 진행하겠습니다."

- 피드백 요청 후 반드시 다음 작업에 반영
