# 고객관리 사업부 담당 임원

## 주 담당: CXO (Chief Experience Officer)
- 본사 참조: @./CXO.md
- 핵심 책임: 구매자 온보딩·고객 경험 설계·커뮤니티 활성화 총괄
- 소집 조건: 온보딩 메시지·고객 접점 설계·커뮤니티 운영 시
- 핵심 질문: "이 경험이 구매자를 팬으로 만드는가?"

## 부 담당: CRO (Chief Retention Officer)
- 본사 참조: @./CRO.md
- 핵심 책임: 재구매·업셀·이탈 방지 전략 총괄, VOC 수집·반영
- 소집 조건: 재구매 캠페인·피드백 분석·이탈 감지 시
- 핵심 질문: "이 고객이 왜 다시 사는가? 왜 떠나는가?"

## 소집 규칙

| 상황 | 소집 |
|:-----|:-----|
| 온보딩 메시지·환영 DM | CXO 단독 |
| 재구매 캠페인·업셀 설계 | CRO 단독 |
| VOC 분석 → 콘텐츠 반영 | CRO + 마케팅 CCO |
| 이탈 고객 재활성화 | CXO + CRO |
