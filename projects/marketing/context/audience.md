# 메인 타겟 — AI 초보자·직장인

## 타겟 프로파일

| 항목 | 내용 |
|:-----|:-----|
| 핵심 타겟 | AI를 업무나 일상에 활용하고 싶은 직장인 및 일반인 |
| 연령대 | 25~40대 |
| 상황 | 본업 유지 + AI 활용법 익히고 싶음 |
| 정보 소비 패턴 | 짧고 즉시 써먹을 수 있는 콘텐츠 선호 |

## 핵심 페인포인트

1. "AI 뉴스는 많은데 나한테 실제로 어떻게 쓰는지 모르겠다"
2. "ChatGPT 써봤는데 뭘 물어봐야 할지 모르겠다"
3. "배우고 싶은데 시간이 없다 (직장 병행)"
4. "비싼 강의 결제했다가 시간 낭비한 경험 있음"

## 콘텐츠 제작 시 적용 원칙

- **전문용어 사용 후 반드시 쉬운 설명 추가** — "쉽게 말하면..."
- **Before → After 구조** — 이 정보를 알기 전 vs 후의 차이 명확히
- **즉시 실행 가능한 1가지** — "오늘 당장 해볼 수 있는 것" 1개 포함
- **공포·압박 제거** — "AI에 뒤처진다" 류 불안 자극 금지

## 구매 결정 요인

| 요인 | 설명 |
|:-----|:-----|
| 신뢰 | "이 사람 정보가 실제로 검증됐나?" |
| 효용 | "이걸 사면 바로 써먹을 수 있나?" |
| 가격 | 부담 없는 소액 (4,000~7,000원대) |
| 접근성 | 어렵지 않고 나도 할 수 있겠다는 느낌 |

## 타겟이 아닌 대상

- AI 개발자·엔지니어 (기술 깊이 콘텐츠 불필요)
- AI 업계 종사자 (이미 아는 내용)
