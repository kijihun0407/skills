# 운영 채널 — 인스타그램

## 채널 정의

| 항목 | 내용 |
|:-----|:-----|
| 플랫폼 | 인스타그램 (릴스 중심) |
| 주력 포맷 | 릴스 (트래픽 유입 핵심) + 캐러셀 (전문성 입증) |
| 운영 주기 | 릴스 주 3회 / 캐러셀 주 2회 |
| 보조 포맷 | 스토리 (CTA 리마인드·일상) |

## 포맷별 역할

| 포맷 | 목적 | 성공 지표 |
|:-----|:-----|:----------|
| 릴스 | 신규 유입·도달 확장 | 조회수·공유수 |
| 캐러셀 | 전문성 입증·저장 유도 | 저장수·팔로우 전환 |
| 스토리 | 기존 팔로워 유지·CTA 리마인드 | 링크 클릭·DM 진입 |

## 알고리즘 원칙

- 릴스는 **첫 3초 이탈률**이 핵심 — 후킹 없으면 노출 없음
- 캐러셀은 **슬라이드 넘기기 완료율**이 핵심 — 각 장이 다음 장으로 넘어갈 이유를 줘야 함
- 저장·공유는 도달을 직접 높이는 신호 — CTA는 항상 이 둘 중 하나 우선

## 금지 패턴

- 릴스에 인트로 인사 (첫 3초 낭비)
- 캐러셀 마지막 장 CTA 없음
- 동일 포맷 연속 5회 이상 (포맷 다양성 유지)
