# 절대 금지 행동

> 이 사업부의 모든 작업 전에 아래 4개를 자동 검증합니다.
> 하나라도 해당되면 작업을 중단하고 사용자에게 알립니다.

---

## 절대 금지

1. **경쟁 크리에이터 이름 직접 언급 또는 비교**
   - 특정 AI 크리에이터 계정을 거론하며 비교·비방하는 표현 금지
   - 간접 비교도 포함 ("다른 채널과 달리~" 류)

2. **가격 인하 유인 표현**
   - "최저가", "할인", "세일", "지금만 이 가격" 등 전자책·프롬프트 판매 시 가격 압박 표현 금지
   - 희소성 조작 표현 포함 ("오늘 자정까지만", "선착순 00명")

3. **성과 보장 표현**
   - "이걸 하면 반드시 수익 난다", "100% 팔로워 증가", "무조건 됩니다" 등 AI 활용 효과를 단정하는 표현 금지
   - 수익·성장·자동화 효과에 대한 과장된 약속 포함

4. **검증되지 않은 수치·후기 인용**
   - 출처 없는 조회수, 전환율, 팔로워 성장 수치 인용 금지
   - 직접 검증하지 않은 타인의 후기·사례를 사실처럼 제시 금지

---

## 작업 전 자동 체크

모든 작업(대본·DM 시나리오·세일즈 문구·리포트) 시작 전 실행:

```
[ ] 1. 경쟁 크리에이터 언급·비교 없음?
[ ] 2. 가격 인하·희소성 조작 표현 없음?
[ ] 3. 성과 보장·과장 표현 없음?
[ ] 4. 모든 수치에 출처 또는 직접 검증 근거 있음?
```

**4개 모두 통과 시** → 작업 진행
**1개라도 해당 시** → 작업 중단 + 사용자에게 알림:
> "⚠️ 금지행동 감지: [항목 번호] [내용]. 수정 후 진행하겠습니다."
