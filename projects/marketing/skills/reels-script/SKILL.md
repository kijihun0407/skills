## 3-Phase 파이프라인

### Phase 1. 리서치 (각도 설정)
- 주제 × 타겟(AI 초보자·직장인) 정합성 점검
- 고객 여정 단계 확인 → 감정 목표 설정 (호기심 / 신뢰 / 효능감 중 택1)
- 후킹 각도 3개 생성 → 최강안 선택

### Phase 2. 작성 (3버전 필수)
대본 3개를 **다른 스타일**로 생산:

1. **교육형 × 결과-문제-해결** (30~35초)
   - "이걸 모르면 손해" 구조 → 결과 먼저 보여주고, 문제 제시, 해결책 전달
2. **스토리텔링 × 시간순** (35~40초)
   - "나도 처음엔 몰랐는데" 구조 → Before 상황 → 발견 → After 변화
3. **권위형 × 순위매기기** (30초)
   - "AI 툴 TOP 3" 구조 → 리스트 나열로 빠른 가치 전달

각 대본에 포함:
- 후킹 멘트 (첫 3초)
- 본문 스크립트 (씬별 대사/자막)
- 캡션 3버전 (짧게 / 중간 / 해시태그 포함)
- CTA 1개 (저장 / 팔로우 / 링크 클릭 중 택1)
- 발행 추천 시간

### Phase 3. 검증
- [ ] 후킹 3초 체크 통과?
- [ ] 타겟(AI 초보자) 톤앤매너 일치? (전문용어 단독 사용 없음)
- [ ] 금지 행동 위반 없음? (허위 성과·비방·과장)
- [ ] CTA가 명확히 1개인가?

```
[대본 A — 교육형 × 결과-문제-해결]
길이: 00초
후킹 (0~3초): ...
본문:
  씬1: ...
  씬2: ...
  씬3: ...
CTA: ...
캡션1 (짧게): ...
캡션2 (중간): ...
캡션3 (해시태그): ...
발행 추천 시간: ...

[대본 B — 스토리텔링 × 시간순]
...

[대본 C — 권위형 × 순위매기기]
...
```

## 검수 기준

| ID | 기준 | 합격 조건 | 가중치 |
|:---|:-----|:----------|:-------|
| R1 | 후킹 3초 | 질문·충격·반전 중 하나 | 필수 |
| R2 | 길이 | 30~40초 (스타일별 기준 내) | 필수 |
| R3 | CTA 단일 | 정확히 1개 | 필수 |
| R4 | 전문용어 처리 | 단독 사용 없음, 비유 병기 | 필수 |
| R5 | filler 없음 | 인사·감사·"오늘은~" 0개 | 필수 |
| R6 | 정보 검증 | 추측 수치·허위 성과 없음 | 필수 |
| R7 | 3버전 완성 | 스타일 다른 대본 3개 | 필수 |
| R8 | 캡션 3버전 | 각 대본당 캡션 3개 | 권장 |
