# socialbiz-setup — 소셜비즈 DM·댓글 자동화 세팅

## 목적
콘텐츠로 유입된 잠재 고객을 DM 자동화로 리드 수집·전환까지 연결
Before: 수동 DM 응대 / After: 자동화 트리거·시퀀스 완성본

## 입력 (Input)

| 항목 | 필수 여부 | 설명 |
|:-----|:---------|:-----|
| 자동화 목적 | 필수 | `리드 수집` / `전자책 전환` / `스폰서십 문의 수신` 중 택1 |
| 트리거 콘텐츠 | 필수 | 어떤 릴스·댓글에 연결할지 (게시물 주제 또는 키워드) |
| 전환 목표 | 필수 | DM 이후 유도할 행동 (링크 클릭 / 이메일 수집 / 결제 페이지 이동) |
| 현재 시퀀스 | 선택 | 기존 DM 템플릿이 있으면 첨부 |

## 실행 파이프라인

### Phase 1 — 트리거 설계
- 댓글 키워드 트리거 정의 (예: "정보", "원해", "보내줘")
- DM 진입 조건 명확화
- 자동화 범위 결정 (댓글 → DM 자동 / DM 내 시퀀스)

### Phase 2 — 메시지 시퀀스 작성
- 메시지 1 (즉시 발송): 환영 + 가치 제안
- 메시지 2 (행동 유도): 링크·정보 전달
- 메시지 3 (선택, 넛지): 미응답 시 1회 리마인드

### Phase 3 — 운영 규칙 문서화 + 납품
- 소셜비즈 세팅 가이드 (어디에 무엇을 입력하는지)
- 주의사항·금지 패턴 명시

## 검수 기준

| ID | 기준 | 합격 조건 | 가중치 |
|:---|:-----|:----------|:-------|
| S1 | 트리거 키워드 | 1개 이상 명시 | 필수 |
| S2 | CTA 단일 | 메시지당 행동 유도 1개 | 필수 |
| S3 | 톤 | 공격적·조급한 표현 없음 | 필수 |
| S4 | 미검증 수치 없음 | 효과·수익 보장 표현 없음 | 필수 |
| S5 | 운영 가이드 포함 | 소셜비즈 입력 위치 명시 | 필수 |
