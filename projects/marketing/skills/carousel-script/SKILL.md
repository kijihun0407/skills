## 캐러셀 구조 공식

```
슬라이드 1: 후킹 제목 (숫자 포함 필수) ← biggie_ai 스타일
슬라이드 2~5: 핵심 내용 (슬라이드당 1 인사이트, 15자 이내)
슬라이드 6: 실용 팁 / 바로 써먹는 방법 ← sangkyun 스타일
슬라이드 7: CTA (팔로우 / 저장 / 댓글 중 1개만)
```

---

## 3-Phase 파이프라인

### Phase 1. 기획 (각도 설정)
- 주제 × 타겟(AI 초보자·직장인) 정합성 점검
- 고객 여정 단계 → 감정 목표 (호기심 / 신뢰 / 효능감)
- 제목 후보 3개 생성 → 숫자 포함 최강안 선택

### Phase 2. 텍스트 작성
슬라이드별 텍스트 작성 기준:
- **슬라이드 1**: 숫자 포함 제목, 15자 이내 ("AI 툴 TOP 3", "2시간이 10분으로")
- **슬라이드 2~5**: 슬라이드당 1가지 인사이트, 본문 2~3줄
- **슬라이드 6**: 당장 써먹을 수 있는 실용 팁 1개
- **슬라이드 7**: CTA 1개만 ("저장해두고 써보세요" / "팔로우하면 매일 업데이트")

### Phase 3. CSV 출력
Canva Bulk Create 업로드용 CSV를 아래 형식으로 출력:

```csv
slide1_title,slide2,slide3,slide4,slide5,slide6_tip,slide7_cta
"제목","슬라이드2 본문","슬라이드3 본문","슬라이드4 본문","슬라이드5 본문","실용 팁","CTA 문구"
```
