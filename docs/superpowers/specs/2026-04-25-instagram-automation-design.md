# 인스타그램 콘텐츠 자동화 파이프라인 설계

**작성일**: 2026-04-25
**목적**: 릴스 + 캐러셀 제작 자동화 시스템 구축
**기준 채널**: 인스타그램 (릴스 주 3회 + 캐러셀 주 2회)

---

## 1. 전체 파이프라인

```
① 주제 선정
   └─ 도구: Claude (content-research 스킬)
   └─ 입력: 트렌드 키워드, 고객 여정 단계
   └─ 출력: 주제 5개 (릴스 3 + 캐러셀 2)

② 대본 / 캐러셀 텍스트 생성
   ├─ 릴스: Claude (reels-script 스킬) → 대본 3버전 + 캡션
   └─ 캐러셀: Claude (carousel-script 스킬, 신규) → 슬라이드 텍스트 + CSV 출력

③-A 릴스 영상 제작
   └─ 도구: HeyGen (AI 아바타)
   └─ 스타일: @biggie_ai — 비노출형, 단정한 뉴스 속보 톤
   └─ 입력: 대본 붙여넣기 → 아바타 영상 자동 생성
   └─ 출력: 9:16 영상 파일

③-B 캐러셀 디자인 제작
   └─ 도구: Canva Bulk Create
   └─ 입력: Claude가 출력한 CSV → Canva 템플릿에 업로드
   └─ 출력: 슬라이드 5~7장 자동 생성 → 일괄 다운로드

④ 릴스 후편집
   └─ 도구: 캡컷
   └─ 작업: 자막 추가 + 인기 상승 음악 삽입
   └─ 기준: 9:16 / 1080p / 첫 프레임 텍스트 오버레이

⑤ 예약 발행
   └─ 도구: 인스타그램 앱 예약 기능
   └─ 주기: 릴스 월·수·금 / 캐러셀 화·목
```

## 3. AI 아바타 방식 — @biggie_ai 기준

### 스타일 정의
- **비노출형** — 얼굴 노출 없음, 아바타 또는 텍스트 중심 영상
- **톤**: 단정형 종결 ("~됩니다", "~했습니다"), 감탄사 없음
- **구조**: 숫자·통계 전면 배치 → 빠른 템포 → 뉴스레터식 정제
- **후킹**: Hook(0~3초) - Hold(3~25초) - Reward(마지막 5초) 구조
- **길이**: 최대 30초

### HeyGen 작업 흐름
1. Claude → 대본 생성 (30초 이내, @biggie_ai 톤 적용)
2. HeyGen → 대본 붙여넣기 → 아바타 선택 → 영상 생성
3. 영상 다운로드 → 캡컷으로 자막·음악 추가
4. 인스타그램 업로드

## 4. 캐러셀 자동화 — Canva Bulk Create 방식

### CSV 출력 형식 (Claude → Canva)
```
slide1_title, slide2, slide3, slide4, slide5, slide6_cta
"AI 툴 TOP 3", "1. ChatGPT...", "2. Claude...", "3. Perplexity...", "핵심 요약...", "저장하고 나중에 써보세요"
```

### Canva 템플릿 변수 필드
- `{{제목}}` — 슬라이드 1 (후킹 제목, 숫자 포함)
- `{{슬라이드2}}` ~ `{{슬라이드5}}` — 본문 (슬라이드당 1 인사이트)
- `{{실용팁}}` — 슬라이드 6 (바로 써먹는 방법)
- `{{CTA}}` — 슬라이드 7 (팔로우/저장/댓글 중 1개)

### 캐러셀 구조 공식
```
슬라이드 1: 후킹 제목 (숫자 포함) ← biggie_ai 스타일
슬라이드 2~5: 핵심 내용 (슬라이드당 1 인사이트)
슬라이드 6: 실용 팁 / 바로 써먹는 방법
슬라이드 7: CTA (팔로우 / 저장 / 댓글 중 1개만)
```

## 6. 자동화 원칙 (강의 기준)

- **데이터가 쌓여야 자동화가 고도화된다** — 초기 5~10개는 수동 병행
- **대본 먼저, 화면 나중** — 텍스트 품질이 조회수를 결정
- **완벽한 1개보다 괜찮은 10개** — 발행 자체가 목적
- **레퍼런스 분석 기준**: 조회수 3만+, DM 공유 > 좋아요인 콘텐츠 벤치마크
