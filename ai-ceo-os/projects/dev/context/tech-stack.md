# 기술 스택 · 환경 정의 — 개발 사업부

## 현재 기술 스택

| 분류 | 도구 | 용도 |
|:---|:---|:---|
| AI 도구 | Claude Code | 코드 생성 · 자동화 |
| AI 도구 | ChatGPT API | 콘텐츠 보조 |
| 언어 | Python | 자동화 스크립트 |
| 언어 | HTML/CSS | 랜딩페이지 |
| 배포 | 미정 | 추후 확정 |
| 데이터 | Instagram API | 통계 수집 |

## 개발 원칙

- **노코드 우선** — 코딩 없이 해결 가능하면 노코드로
- **직장 병행 고려** — 유지보수 부담 최소화
- **보안 필수** — API 키 환경변수, `.gitignore` 적용

## 환경 설정

- `.env` 파일로 민감 정보 관리
- 로컬 테스트 완료 후 배포
- 개발/프로덕션 환경 분리
