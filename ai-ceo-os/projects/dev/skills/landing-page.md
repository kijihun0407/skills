# 스킬: 랜딩페이지 기획 · 수정

> 전자책 / 프롬프트 판매 페이지 설계 및 개선.

---

## 입력

```
목적: 전자책 판매 / 프롬프트 판매 / 무료자료 수집 중 선택
현재 문제: [이탈률 높음 / 버튼 클릭 안 됨 / 디자인 수정 등]
참고 URL: (있으면)
```

---

## 출력 형식

```markdown
## 페이지 구조
1. 히어로 섹션 — 핵심 메시지 + CTA
2. 문제 제시 — 타겟 고객의 고통
3. 솔루션 — 상품 소개
4. 사회적 증거 — 후기 / 수치
5. 최종 CTA

## 카피라이팅 초안
[각 섹션별 문구]

## 기술 스펙
- 권장 플랫폼:
- 예상 구현 시간:
```

## 품질 기준

| ID | 기준 | 합격 조건 |
|:---|:---|:---|
| L1 | CTA 명확성 | 버튼 문구가 행동을 명확히 지시 |
| L2 | 모바일 최적화 | 인스타 유입 → 모바일 우선 |
| L3 | 로딩 속도 | 3초 이내 |
| L4 | 가격 노출 | 4,000원 / 7,000원 명확히 표기 |
