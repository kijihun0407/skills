1. 생산자 3원칙 자가 답변 (PLAN.md §8)
═════════════════════════════════
▶ 원칙 1 — 문제 먼저: "누가 돈 내고 쓸 문제인가?"
    답: AI 정보는 매일 보지만 실행은 0인 비전공자/초보자.
        유튜브·릴스 저장만 하고, ChatGPT 번역기로만 쓰는 사람들.
        "도대체 AI로 뭐부터 어떻게 시작하지?"라는 막막함이 진짜 문제다.
        → 돈 내고 쓸 만한 문제 ✅ (실행 0 → 실행 가능으로 전환)

▶ 원칙 2 — 가치 먼저: "다시 올 이유가 있는가?"
    답: 무료 프롬프트 5개를 복붙 즉시 실무에 써봤을 때 "된다"는 경험을 제공.
        실제로 써보면 인스타 팔로우 + 이메일 구독으로 자연스럽게 이어진다.
        "추가 프롬프트 3개는 인스타에서" 구조로 재방문 이유를 설계했다.
        → 다시 올 이유 있음 ✅ (무료 경험 → 신뢰 → 재방문/재구매)

▶ 원칙 3 — 수익 먼저: "좋아요 100 vs 결제 1"
    내 첫 결제 시나리오: D+30까지 10만원
      - 무료 프롬프트 다운로드 완료 직후 묶음팩 7,000원 결제 페이지 즉시 노출
      - 토스페이먼츠 연동 → 원클릭 결제
      - 목표: D+30 이메일 100명 × 전환 15% = 15건 × 7,000원 = 105,000원
      → 좋아요보다 결제 1건 우선 구조 설계 완료 ✅

  "결제 플로우 먼저 검증 → 이메일 수집 → UI 다듬기"로 바꿨다.
  팔로워 늘리는 것보다 D+30 결제 10만원이 먼저다.
