1. INTERVIEW.md 자가 인터뷰 결과
   ═════════════════════════════════
   사용 프롬프트: bootcamp-kit/prompts/01-interview-setup.md

▶ Q1. 누구를 위한 서비스인가?:
  AI 활용을 원하고 관심을 가지는 사람. 나이대·직군 무관.
  공통점: 정보는 많이 봤지만 실행은 0, 전문용어에 지레 포기하는 비전공자/초보자.

▶ Q2. 그 사람이 지금 겪는 가장 큰 문제는?:

1. 실행의 막막함 — AI 정보는 매일 보는데 내 컴퓨터에서 뭐부터 시작할지 모름
2. 정보 과부하와 두려움 — 코딩/전문용어만 나와도 지레 겁먹고 포기
3. 겉핥기식 활용 — ChatGPT에 단순 질문만, 실무 적용/수익화로 한 발짝도 못 나감

▶ Q3. 지금 어떻게 해결하고 있나?:

1. 수집 강박형 — 릴스/유튜브 저장만 함, 실행 없음
2. 강의 의존형 — 비싼 강의 구매 → 코딩 용어에 중간 포기
3. 무작정 결제형 — ChatGPT Plus 결제 → 프롬프트 몰라서 번역기로만 사용

▶ Q4. 현재 해결책의 단점은?:

1. 실질적인 내 결과물이 0 (눈팅의 한계)
2. 비전공자 현실 무시한 진입장벽
3. 비용 대비 극악의 효율 (돈 낭비 후 원래대로)

▶ Q5. 내 서비스만의 차별점은?:
  "보는 AI에서 쓰는 AI로 — 복붙 즉시 내 업무에 적용되는 프롬프트"

- 전문용어 없음, 열자마자 쓸 수 있는 형태만 제공
- 비전공자 전용 실무 시나리오 포함
- 무료 먼저 경험 → 신뢰 쌓인 후 결제

▶ Q6. 핵심 기능 3개는?:

1. 즉시 다운로드 — 이메일 입력 → 무료 프롬프트 5개 PDF 즉시 전달 + Before/After 비교
2. 원클릭 업셀 — 다운로드 완료 즉시 묶음 7,000원 결제 페이지 노출
3. 팔로우 유도 — "추가 프롬프트 3개는 인스타에서" → 팔로워 동시 성장

▶ Q7. 무료 vs 유료, 가격은?:

- 무료: 실무 프롬프트 5개 PDF (이메일 수집)
- 유료 A: 프롬프트팩 4,000원
- 유료 B: 전자책 4,000원
- 유료 묶음 (메인): 전자책 + 프롬프트팩 7,000원

▶ Q8. 수익화 경로는?:
  릴스 도달 → 링크인바이오 → 무료 다운로드(이메일 수집) → 즉시 업셀 + 3일 자동 팔로업
  → 묶음 7,000원 결제 → 파일 자동 전달

▶ Q9. 3개월 후 모습은?:

- 이메일 리스트 2,000명
- 월 결제 143건 이상
- 월 매출 100만원 이상
- 랜딩 → 다운로드 전환율 30%+

▶ Q10. 실패 정의는?:

- D+30: 이메일 100명 미달 → CTA/랜딩 수정
- D+60: 첫 결제 10건 미달 → 가격/상품 피벗
- D+90: 월 매출 30만원 미달 → 서비스 구조 전면 재검토

2. 완성된 PLAN.md 핵심 섹션
═════════════════════════════════
▶ §1 한 줄 정의:
  "AI를 쓰고 싶은 비전공자가 무료 프롬프트를 받고, 자동으로 유료 묶음팩까지 구매하는 랜딩페이지"

▶ §2 타겟:
  AI 활용에 관심 있는 비전공자/초보자. 나이대·직군 무관.
  정보는 많이 봤지만 실행은 0, 전문용어에 지레 포기하는 사람.

▶ §3 해결 (Before/After):
  Before: 유튜브/릴스 저장만, 비싼 강의 중간 포기, ChatGPT 번역기로만 사용
  After: 5분 안에 무료 프롬프트 5개 다운로드 → 복붙 즉시 업무 적용 → 신뢰 생겨서 7,000원 자연스럽게 결제

▶ §4 핵심 기능 3개:

1. 즉시 다운로드 + Before/After 신뢰 형성
2. 원클릭 업셀 (토스페이먼츠 연동)
3. 팔로우 유도 장치 (이메일 + 팔로워 동시 성장)

▶ §5 차별점:
  복붙 즉시 실행 / 비전공자 전용 시나리오 / 무료 먼저 → 신뢰 후 결제

▶ §7 수익화 경로:
  릴스 → 랜딩페이지 → 무료 다운로드 → 자동 이메일 → 묶음 7,000원 결제
  (이메일 2,000명 × 전환 7% = 140건 × 7,000원 = 월 98만원)
