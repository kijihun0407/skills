# 보안 체크리스트 — 개발 사업부

> 모든 코드 산출물에 자동 적용.

## 필수 체크

- [ ] API 키 하드코딩 없음 (`.env` 또는 환경변수 사용)
- [ ] `.gitignore`에 `.env` 포함 확인
- [ ] 외부 사용자 입력값 검증 (랜딩페이지 폼 등)
- [ ] 의존 패키지 출처 확인 (공식 패키지만)

## 배포 전 체크

- [ ] 로컬에서 테스트 완료 후 배포
- [ ] 프로덕션 API 키와 개발 API 키 분리
- [ ] 민감 정보 로그 출력 없음

## 금지

- API 키, 비밀번호를 코드에 직접 작성
- 검증 안 된 외부 패키지 즉시 프로덕션 적용
- 보안 체크 없이 사용자 데이터 수집
