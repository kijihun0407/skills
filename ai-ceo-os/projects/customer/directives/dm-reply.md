# Directive: DM 응대 SOP (고객 사업부)

> Layer 1. CXO가 실행 지휘. 팔로워를 팬으로 전환하는 경험을 설계한다.

## 목표

DM 응대를 통해 팔로워를 무료자료(4단계) → 전자책 구매(5단계)로 자연스럽게 이동시킨다.

## 입력값

- DM 내용 (유형 분류 필요)
- 발신자 상태 (팔로워/비팔로워, 기구매자 여부)

## DM 유형 분류 + 응대 방식

| 유형 | 판단 기준 | 응대 방식 |
|:---|:---|:---|
| 무료자료 요청 | "무료자료", "링크", "자료" 키워드 | 즉시 링크 제공 + 전자책 자연스러운 언급 1회 |
| 구매 문의 | "가격", "구매", "전자책" 키워드 | 정보 제공 → 가치 강조 → 가격 압박 금지 |
| 일반 응원/질문 | 기타 | 친근하게 답변 + 팔로우 감사 |
| 불만/항의 | 부정적 감정 | 공감 먼저 → 해결책 → 에스컬레이션 여부 판단 |

## 응대 원칙

- 응답 시간: 24시간 이내
- 톤: 친근하되 프로페셔널
- 가격 압박 절대 금지 — 정보 제공으로 신뢰 먼저
- 전자책 언급은 자연스럽게 1회만 (무료자료 요청 시)

## 출력 형식 (응대 템플릿)

```
[무료자료 요청]
안녕하세요! 반가워요 😊
무료자료 링크 드릴게요: {링크}
혹시 더 깊이 배우고 싶으시면 전자책도 있어요 → {링크}

[구매 문의]
안녕하세요! 문의 감사해요 🙏
{상품명}은 {핵심 가치}를 드리는 자료예요.
가격은 {가격}이고, 여기서 확인하실 수 있어요 → {링크}
```

## 엣지케이스

- 동일 질문 3회 이상 수신 → DM 응대 SOP에 FAQ 항목 추가
- 불만 고객 → 즉시 공감 응대, 해결 안 되면 CEO에게 에스컬레이션
