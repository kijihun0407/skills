# Directive: DM 자동화 셋팅 SOP (소셜비즈)

> Layer 1. 댓글 → 자동 DM 발송 시스템 구축 및 운영.

## 목표

릴스 댓글 키워드 → 자동 DM 발송으로 리드 수집 및 무료자료→유료 전환율 극대화.

## DM 자동화가 중요한 이유

1. **전환율 극대화** — 댓글 남긴 관심 있는 시청자에게 즉시 자료/링크 전달
2. **리드 수집** — DM으로 연락처/이메일 자동 수집
3. **팬덤 구축** — "무료 자료 보내드려요" → 신뢰 → 유료 전환

## 도구 선택

| 도구 | 특징 | 가격 | 추천 |
|:---|:---|:---|:---|
| **소셜비즈** (1순위) | 한국형, 한국어 UI | 유료 | 한국 사용자 1순위 |
| **ManyChat** | 글로벌 1위, 기능 풍부 | 무료(1000명)/Pro $15/월 | 해외 레퍼런스 풍부 |
| **Chatfuel** | GPT-4 연동 자연어 응대 | $19.99/월~ | AI 챗봇 원할 때 |

## 소셜비즈 셋팅 순서

1. 릴스 업로드 준비 완료 (마케팅 directives 참조)
2. 소셜비즈 접속 → 새 캠페인 생성
3. 트리거 키워드 설정 (예: "자료", "링크", "보내줘")
4. 자동 DM 내용 작성 (아래 템플릿 참조)
5. 자동 댓글 내용 작성 (있을 경우)
6. 릴스 업로드 직후 소셜비즈 게시 활성화

## DM 자동화 메시지 템플릿

```
안녕하세요! 반가워요 😊
{릴스 제목} 무료자료 드릴게요!
→ {링크}

혹시 더 깊이 배우고 싶으시면
전자책도 있어요 → {링크}

궁금한 점 있으시면 언제든 DM 주세요!
```

## 운영 주의사항

- 댓글이 너무 많이 달리면 소셜비즈에서 패널티 방지 자동 중단됨
- 어뷰징 메일 수신 시 = 콘텐츠 반응이 좋다는 의미이기도 함
- 매 릴스마다 복사 후 제목/링크만 바꿔 재사용 → 첫 셋팅 후 매우 수월

## Self-Annealing

- DM 응답률 낮으면 → 메시지 문구 A/B 테스트 → 더 나은 버전으로 교체
- 새로운 자주 묻는 키워드 발견 → 트리거 키워드 추가
- `dm-reply.md` 에 패턴 업데이트
