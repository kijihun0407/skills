# Directive: 프로필 링크 셋팅 SOP

> Layer 1. 인스타그램 프로필 링크 = 수익의 입구. 팔로워를 고객으로 전환하는 관문.

## 목표

프로필 링크 하나로 팔로워를 커뮤니티 → 상품 → 채널로 효율적으로 연결한다.

## 반드시 포함해야 할 5가지

| # | 항목 | 이유 |
|:---|:---|:---|
| 1 | **비즈니스 이메일** | 광고 제안/협업 문의 창구 (필수) |
| 2 | **커뮤니티 링크** (카톡 오픈채팅) | 팔로워 → 커뮤니티 멤버 전환 |
| 3 | **상품/서비스 링크** | 직접 매출 발생 경로 |
| 4 | **다른 채널** (유튜브, 카페 등) | 크로스 플랫폼 팬 확보 |
| 5 | **무료 자료 다운로드** | 리드 수집 (이메일/연락처 확보) |

## 추천 플랫폼

| 플랫폼 | 특징 | 가격 |
|:---|:---|:---|
| **리틀리** | 한국형, 클릭 데이터 제공 | 무료 |
| **링크트리** | 글로벌 1위, 클릭 분석 | 무료 |
| **비틀리** | URL 단축 + 클릭 추적 | 무료 |

> 클릭 데이터 분석으로 어떤 링크가 가장 많이 클릭되는지 파악 → 수요 확인.

## 프로필 링크 구성 예시

```
[리틀리/링크트리 구성]

1. 🎁 무료 AI 활용 가이드 받기 → {무료자료 링크}
2. 💬 AI 실전 단톡방 입장 → {오픈채팅 링크}
3. 📚 전자책 구매 → {상품 링크}
4. 📺 유튜브 채널 → {유튜브 링크}
5. 📧 비즈니스 문의 → {이메일}
```

## 수익화 연동 구조

```
인스타 프로필 방문
→ 프로필 링크 클릭
→ 무료자료 링크 → 이메일 수집 → 유료 상품 안내
→ 커뮤니티 링크 → 오픈채팅 입장 → 신뢰 형성 → 구매
→ 상품 링크 → 직접 구매
```

## 성과 모니터링

- 주 1회 각 링크 클릭 수 확인
- 클릭 많은 링크 → 상단 배치
- 클릭 없는 링크 → 문구 수정 또는 제거
