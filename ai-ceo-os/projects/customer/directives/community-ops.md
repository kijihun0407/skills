# Directive: 커뮤니티 운영 SOP

> Layer 1. 팔로워를 한 곳에 모아 소비자로 전환하는 구조를 만든다.
> 소비자를 미리 모아두면 추후 거래가 훨씬 수월해진다.

## 목표

카카오톡 오픈채팅방 + 네이버 카페 + 카카오톡 채널 3종 세트를 운영해 팔로워를 팬덤으로 전환한다.

## 3가지 커뮤니티 역할

| 채널 | 역할 | 장점 | 운영 목적 |
|:---|:---|:---|:---|
| **카카오톡 오픈채팅방** | 소통 (관계 형성) | 진입 장벽 최저, 즉시 소통 | 실시간 정보 공유, 친밀감 |
| **네이버 카페** | 정보 허브 (신뢰 구축) | 검색 유입, 콘텐츠 축적 | 자료 아카이빙, SEO 유입 |
| **카카오톡 채널** | 세일즈 (전환) | 1:다 메시지, CRM 가능 | 공지, 프로모션, 세일즈 |

## 신영선의 커뮤니티 운영 3원칙

| 원칙 | 설명 | 실천 방법 |
|:---|:---|:---|
| **1. 존재 목적 명확히** | "이 커뮤니티는 왜 존재하는가" 한 줄 답 | 입장 공지에 목적 명시 |
| **2. 운영자도 지킬 원칙** | 운영자가 먼저 규칙을 지켜야 멤버도 따른다 | 매일 정해진 시간 정보 공유, 24h 이내 응답 |
| **3. 기버(Giver) 문화** | "받기 전에 먼저 주는" 문화 | 운영자가 먼저 무료 자료/인사이트 꾸준히 공유 |

> 커뮤니티 인원보다 질이 중요하다. 5명이 서로 성장 응원하는 방 > 100명 중 1~4명만 얘기하는 방.

## 오픈채팅방 운영 프로세스

```
1. 릴스 CTA에 오픈채팅 입장 유도 ("단톡방 링크 댓글에 '링크' 적어주세요")
↓
2. DM 자동화로 오픈채팅 링크 발송
↓
3. 입장 공지 (목적, 운영 원칙, 인사)
↓
4. 매일 AI 트렌드 정보 1개 공유 (운영자 먼저)
↓
5. 질문/피드백 유도
↓
6. 상품 출시 시 커뮤니티 우선 공개 + 특별 할인
```

## 커뮤니티 → 수익 전환 구조

```
인스타 릴스 (1~3단계 팔로워)
→ CTA: "단톡방 입장하세요"
→ 오픈채팅방 (신뢰 형성)
→ 카카오톡 채널 (세일즈 메시지)
→ 무료자료 → 전자책 구매 (5단계)
→ 재구매 + 팬덤 (6단계)
```

## Self-Annealing

- 채팅방 조용해지면 → 새 질문/이벤트로 활성화 시도 → 패턴 기록
- 자주 나오는 질문 → 카페에 FAQ 아카이빙 → DM 응대 SOP 업데이트
