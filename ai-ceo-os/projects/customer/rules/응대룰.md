# 응대 룰 — 고객 사업부

> 팔로워 DM · 댓글 응대 시 자동 적용.

## 기본 원칙

- **응답 시간**: 24시간 이내
- **톤**: 친근하되 프로페셔널 — 반말 금지, 과도한 이모지 금지
- **길이**: 3~5줄 이내 (짧고 명확하게)

## 상황별 규칙

### 무료자료 요청 DM
- 링크 즉시 제공
- 전자책 언급은 자연스럽게 1회만 — "혹시 더 깊이 파고 싶으시다면..." 형태
- 가격 직접 언급 금지

### 구매 문의 DM
- 상품 핵심 가치 1줄 설명 후 링크 안내
- "지금 사세요!" 류 압박 금지
- 추가 질문 환영 멘트 포함

### 콘텐츠 관련 질문
- 2~3줄로 핵심만 답변
- 관련 콘텐츠/자료 있으면 링크 1개 안내
- 모르는 내용은 "확인 후 답변드릴게요" (추측 답변 금지)

### 부정적 반응 / 클레임
- 방어적 답변 금지
- "말씀 감사해요, 더 좋은 콘텐츠로 보답할게요" 수용 후 종료
- 논쟁 절대 금지

## 절대 금지

- 가격 압박 / 할인 제안
- 검증 안 된 AI 정보 DM으로 전달
- 다른 크리에이터 비방
- 이모지 3개 초과 사용
