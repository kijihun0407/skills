# 스킬: DM 응대 템플릿

> 상황별 DM 답장 즉시 산출.

---

## 입력

```
상황: 무료자료 요청 / 구매 문의 / 콘텐츠 질문 / 일반 반응 중 선택
DM 원문: [팔로워가 보낸 메시지]
```

---

## 상황별 템플릿

### A. 무료자료 요청

```
안녕하세요! 관심 가져주셔서 감사해요 :)
[자료명] 여기서 받으실 수 있어요 → [링크]

혹시 AI 활용을 더 깊이 파고 싶으시다면
전자책도 준비되어 있으니 프로필 참고해보세요!
```

### B. 구매 문의

```
안녕하세요! [상품명]에 관심 가져주셔서 감사해요.
[상품 핵심 가치 1줄]
구매는 프로필 링크에서 바로 하실 수 있어요 → [링크]
궁금한 점 있으시면 편하게 질문해주세요!
```

### C. 콘텐츠 질문

```
좋은 질문이에요!
[답변 — 2~3줄, 구체적으로]
더 자세한 내용은 [관련 콘텐츠/자료]에서 확인하실 수 있어요.
```

### D. 일반 반응 (좋아요·감사 DM)

```
감사해요! 도움이 됐다니 정말 뿌듯합니다 :)
앞으로도 써먹을 수 있는 AI 팁 계속 올릴게요.
팔로우하고 놓치지 마세요!
```

## 품질 기준

| ID | 기준 | 합격 조건 |
|:---|:---|:---|
| D1 | 친근한 톤 | 딱딱하지 않고 자연스러운 구어체 |
| D2 | CTA 1개 | 링크 또는 팔로우 중 1개만 |
| D3 | 가격 압박 없음 | "지금 사세요!" 류 금지 |
| D4 | 3줄 이내 | 짧고 읽기 쉽게 |
