## 이 사업부의 미션

릴스 · 카드뉴스로 팔로워를 고객 여정 1→3단계까지 끌어올린다.
궁극 목표: 팔로워 4만 + 무료자료 다운로드 유도 → 유료 상품 전환 연결.

## 핵심 업무 TOP 3

1. **릴스 대본 작성** — 주 3회 발행 기준, 30~40초 완성본
2. **카드뉴스 기획** — 릴스 주제를 5~7장 슬라이드로 재구성
3. **콘텐츠 캘린더 관리** — 주간 발행 스케줄 + 성과 예측

## 콘텐츠 전략

### 얼굴 노출 vs 비노출

| 구분 | 장점 | 단점 | 편집 방식 |
|:---|:---|:---|:---|
| **얼굴 노출형** | 퍼스널 브랜딩 강력, 팬덤 형성 빠름 | 촬영 리소스 높음 | 토킹헤드 + 화면 시연 |
| **얼굴 비노출형** | 부담 적음, 직장인 가능, 위임 쉬움 | 브랜딩 약함 | 텍스트 오버레이 + 화면녹화 + AI 아바타 |

### 도구 스택 선택 (의사결정 트리)

| 상황 | 추천 스택 |
|:---|:---|
| **왕초보, 예산 0** | CapCut(무료) + Canva(무료) + 소셜비즈 |
| **왕초보, 월 3~5만원** | CapCut Pro + Canva Pro + 소셜비즈 |
| **편집 시간 단축 필요** | CapCut + Submagic($20) + Gling.ai($12) |
| **기획 시간 단축 필요** | AI 프롬프트 + Predis.ai($29) |
| **멀티플랫폼 확장** | Opus Clip($19) + Repurpose.io($29) |

> 상세 도구 목록: `../../knowledge/ai-tools.md`

## 3-레이어 아키텍처 (마케팅 사업부)

| 레이어 | 마케팅 적용 | 위치 |
|:---|:---|:---|
| **Layer 1: Directive** | 릴스 제작 SOP, 콘텐츠 캘린더 SOP | `directives/` |
| **Layer 2: Orchestration** | CMO/CCO가 Directive 읽고 실행 지휘 | AI (김이사) |
| **Layer 3: Execution** | 인스타 통계 수집, 성과 계산 스크립트 | `../../execution/` |

### 도구 우선 확인 원칙
작업 시작 전 `../../execution/`에 재사용 가능한 스크립트가 있는지 먼저 확인.
없으면 새 스크립트 작성 제안 후 진행.

### Self-Annealing
- API 에러 / 데이터 불일치 발생 → 스크립트 수정 → 해당 `directives/` SOP 업데이트
- 성공 패턴 발견 → "WIN:" 태그로 `../../tasks/lessons.md`에 기록
