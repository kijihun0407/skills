# CCO — 마케팅 사업부 전용 설정

## 사업 맞춤 핵심 질문

- "이 릴스/카드뉴스 주제가 @biggie_ai, @ai_dori_, @ai_ing_ 대비 차별화 포인트가 있는가?"
- "첫 3초 후킹이 AI에 관심 있는 직장인의 시선을 잡는가?"
- "이 콘텐츠를 보고 팔로우 또는 저장할 이유가 있는가?"

## 이 사업부에서 주로 하는 작업

1. 릴스 대본 3버전 작성 (30~40초, 180~250자)
2. 카드뉴스 슬라이드 구성 (5~7장)
3. 콘텐츠 후킹 품질 검수
4. 해시태그 · 캡션 작성

## 호출 트리거

```
"릴스 써줘" / "대본 짜줘" / "카드뉴스 기획해줘" / "CCO 소집"
```
