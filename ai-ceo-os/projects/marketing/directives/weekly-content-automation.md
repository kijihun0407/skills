## 주간 자동화 파이프라인

### 📅 월요일 (15분) — 기획 + 생성

```
① Claude: "이번 주 AI 트렌드 기반 릴스 3개 + 캐러셀 2개 주제 뽑아줘"
   └─ 출력: 주제 5개 (릴스 3 + 캐러셀 2) + 각 타겟 단계

② Claude: reels-script 스킬로 릴스 대본 3개 생성 (배치)
   └─ 출력: 대본 3버전 + 캡션 3버전 / 릴스당

③ Claude: carousel-script 스킬로 캐러셀 CSV 2개 생성
   └─ 출력: CSV 파일 2개 → .tmp/ 저장
```

### 📅 월요일 (5분) — 캐러셀 디자인

```
④ Canva Bulk Create: CSV 2개 업로드 → 슬라이드 자동 생성
   └─ 상세: directives/canva-bulk-create-sop.md 참조
   └─ 다운로드: PNG 7장 × 2세트
```

### 📅 화요일 (15분) — 릴스 제작

```
⑤ 릴스 영상 제작 (화면녹화 방식 — 비노출형)
   ├─ 방법 A: AI 툴 화면녹화 + 나레이션 (맥 기본 화면 녹화)
   ├─ 방법 B: 캡컷에서 텍스트/움짤 조합 (얼굴 완전 비노출)
   └─ 방법 C: Canva 슬라이드 애니메이션 → 동영상 내보내기

⑥ 캡컷 후편집
   └─ 상세: directives/reels-editing.md 참조
   └─ 체크: 자막 / 음악 / 9:16 / 워터마크 제거
```

### 📅 화요일 (5분) — 예약 발행 세팅

```
⑦ 인스타그램 앱 예약 발행
   ├─ 릴스: 월(09:00) / 수(19:00) / 금(19:00)
   └─ 캐러셀: 화(12:00) / 목(19:00)
```

---

## 릴스 제작 방식 (비노출형 3가지)

### 방법 A: AI 툴 화면녹화 (추천)
```
1. Claude / ChatGPT 화면 녹화 시작
2. 대본 읽으며 AI 툴 실시간 조작
3. 캡컷으로 자막 + 음악 추가
4. 결과: 실용적, 신뢰감 높음
```

### 방법 B: 텍스트 + 움짤 조합
```
1. 캡컷에서 새 프로젝트
2. GIF/움짤 배경 삽입 (tenor.com)
3. 텍스트 애니메이션으로 핵심 문장 순차 등장
4. 음악 + 효과음 추가
5. 결과: 제작 빠름, 클린한 느낌
```

### 방법 C: Canva 슬라이드 → 동영상
```
1. Canva에서 릴스용 슬라이드 제작 (9:16)
2. 프레젠테이션 → 자동 재생 설정
3. 동영상으로 내보내기 (MP4)
4. 캡컷으로 음악 + 자막 추가
5. 결과: 디자인 일관성 높음, 캐러셀과 비주얼 연동 가능
```
