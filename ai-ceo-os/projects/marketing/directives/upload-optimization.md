# Directive: 업로드 최적화 SOP (2026)

> Layer 1. 편집 완료 후 인스타그램 업로드 시 반드시 적용할 5가지 설정.

## 업로드 전 5가지 필수 설정

### 1. 고화질 업로드 설정
인스타 앱 > 설정 및 개인정보 보호 > 데이터 사용 및 미디어 품질 > **"최고 화질로 업로드" ON**
(인스타 공식 발표: 고화질 업로드가 알고리즘 노출에 유리)

### 2. 프로페셔널 계정 전환
크리에이터 계정으로 전환 → 데이터 분석, 공지 채널 등 기능 활성화

### 3. 업로드 시간
**평일 퇴근 시간대 (오후 7시~9시)** — 핸드폰 사용량 최고점 → 더 많은 도달

### 4. 인기 상승 음악 활용
인스타 앱에서 "인기 상승" 태그 음악 사용 → 알고리즘 노출 + 시청지속시간 유리

### 5. 설문 기능 활용
콘텐츠 인게이지먼트 지수 활성화 → 시청자 참여 유도 (스토리 또는 게시물)

## 2026 추가 최적화 팁

| 항목 | 내용 | 중요도 |
|:---|:---|:---|
| **자막 필수** | 무음 시청 증가 → 자막 없으면 도달 20~30% 감소 | 필수 |
| **워터마크 제거** | TikTok/CapCut 워터마크 → 알고리즘 억제 | 필수 |
| **Trial Reels** | 비팔로워에게만 24시간 테스트 공개 → 성과 판단 후 정식 발행 | 권장 |
| **저장(Save) 유도** | "이 글 저장해두세요" — 좋아요보다 저장률이 더 강한 알고리즘 시그널 | 필수 |
| **협업(Collab) 포스트** | 두 계정 팔로워 모두에게 도달, 알고리즘 가중치 부여 | 선택 |
| **오리지널 콘텐츠** | 동일 콘텐츠 3개 플랫폼 크로스 포스팅은 2026년 효과 없음 | 주의 |

## 소셜비즈 DM 자동화 연동 순서

1. 릴스 업로드 준비 완료
2. 소셜비즈 설정 저장 (트리거 키워드 설정)
3. 릴스 업로드 직후 소셜비즈 게시 활성화

> 상세 DM 자동화 SOP: `../../customer/directives/dm-automation.md`

## 업로드 완료 후

- `outputs/{연도}/{월}/{일}/[릴스]_제목.md` 에 발행 기록
- 발행 후 24~48시간 내 Trial Reels 성과 확인
- 반응 좋으면 → 정식 피드에 고정 검토
