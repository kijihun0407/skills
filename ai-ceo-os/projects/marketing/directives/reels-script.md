# Directive: 릴스 대본 작성 SOP (마케팅 사업부)

> Layer 1. CCO + CMO가 실행 지휘. 검증된 절차만 따른다.

## 목표

주 3회 발행 기준, 30~40초 릴스 완성본 대본 제작.

## 입력값

- 주제 (content-research 결과 또는 CEO 직접 지정)
- 타겟 고객 여정 단계 (1~6단계 중 하나)
- 발행 예정일

## 실행 순서

1. `skills/reels-script.md` 스킬 파일 참조
2. 고객 여정 단계에 맞는 후킹 방식 선택
   - 1단계(호기심): 충격 / 반전 오프닝
   - 2단계(신뢰): 데이터 / 증거 오프닝
   - 3단계(자신감): "이거 당신도 할 수 있어" 공감 오프닝
3. 대본 초안 작성
4. 품질 체크리스트 자동 검수
5. 파일 저장: `outputs/{연도}/{월}/{일}/[릴스]_제목.md`

| ID | 기준 | 합격 조건 | 가중치 |
|:---|:---|:---|:---|
| Q1 | 첫 문장 후킹 | 3초 내 시선 잡는 문장 | 필수 |

## 엣지케이스

- 주제가 너무 기술적이면 → "쉽게 말하면..." 구간 의무 삽입
- 트렌드 정보가 오래됐으면 → CEO에게 최신 데이터 요청
- 유사 대본이 이미 발행됐으면 → 앵글 변경 후 재작성
