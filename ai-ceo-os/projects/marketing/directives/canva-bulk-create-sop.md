# Directive: Canva Bulk Create 캐러셀 자동화 SOP

> Layer 1. Claude(carousel-script) → CSV → Canva Bulk Create → 슬라이드 완성
> Canva 계정만으로 캐러셀 주 2회 제작을 반자동화한다.

---

## 전체 흐름

```
① Claude에게 "캐러셀 만들어줘 + 주제" 입력
        ↓
② carousel-script 스킬 실행 → CSV 출력
        ↓
③ Canva 템플릿에 CSV 업로드 (Bulk Create)
        ↓
④ 슬라이드 7장 자동 생성 → 일괄 다운로드
        ↓
⑤ 인스타그램 업로드 + 캡션 붙이기
```

---

## STEP 1. Claude에서 CSV 뽑기

**입력 예시:**
```
캐러셀 만들어줘. 주제: "직장인이 당장 쓸 수 있는 ChatGPT 프롬프트 5개", 타겟 단계 2단계(신뢰)
```

**출력 확인 사항:**
- [ ] `slide1_title` ~ `slide7_cta` 7개 컬럼 포함
- [ ] 큰따옴표로 각 셀 감싸져 있음
- [ ] 쉼표 포함 텍스트도 깨지지 않음

**CSV 저장:**
파일명: `carousel_YYYYMMDD_주제.csv`
저장 위치: `ai-ceo-os/.tmp/`

---

## STEP 2. Canva 템플릿 세팅 (최초 1회만)

> 한 번만 만들면 매번 재사용. 최초 세팅에 20~30분 소요.

### 2-1. 캐러셀 템플릿 선택
1. Canva 접속 → 새 디자인 → `Instagram 게시물 (정사각형)` 또는 `4:5 (1080x1350)`
2. 템플릿 탭 → "카드뉴스" 또는 "정보형 인포그래픽" 검색
3. **7페이지짜리 템플릿** 선택 (없으면 1페이지 템플릿을 7번 복제)

### 2-2. 변수 필드 설정 (핵심)
각 슬라이드의 텍스트 요소를 클릭 → **변수명**으로 대체

| 슬라이드 | 변수명 | 내용 |
|:---------|:-------|:-----|
| 1장 (제목) | `slide1_title` | 숫자 포함 후킹 제목 |
| 2장 | `slide2` | 인사이트 1 |
| 3장 | `slide3` | 인사이트 2 |
| 4장 | `slide4` | 인사이트 3 |
| 5장 | `slide5` | 인사이트 4 |
| 6장 (실용팁) | `slide6_tip` | 당장 써먹을 팁 |
| 7장 (CTA) | `slide7_cta` | 팔로우/저장/댓글 중 1개 |

**변수 입력 방법:**
1. 텍스트 박스 선택
2. 상단 메뉴 → `앱` → `Bulk Create` 클릭
3. `데이터 연결` → 해당 텍스트에 변수명 매핑

### 2-3. 디자인 고정 요소 (변수 아닌 것)
- 배경색 / 폰트 / 브랜드 로고 → **고정 유지**
- `@ai.saver_` 계정명 → 매 슬라이드 하단 고정 텍스트로 배치
- 브랜드 컬러: 통일된 1~2색 팔레트 사용

---

## STEP 3. Bulk Create 실행

1. Canva 상단 → `앱` → `Bulk Create` 클릭
2. `CSV 업로드` → `.tmp/carousel_YYYYMMDD.csv` 선택
3. 컬럼명 ↔ 변수명 자동 매핑 확인
   - `slide1_title` → 1장 제목 텍스트
   - `slide2` → 2장 텍스트 … (순서대로)
4. `생성` 클릭 → 7장 슬라이드 자동 완성

---

## STEP 4. 다운로드 + 업로드

**다운로드:**
- `공유` → `다운로드` → `PNG` (고화질) 또는 `JPG`
- 7장 전체 선택 → ZIP으로 일괄 다운로드

**업로드:**
- 인스타그램 앱 → 새 게시물 → 여러 장 선택 → 순서대로 7장 업로드
- 캡션: Claude가 생성한 캡션 3버전 중 선택 붙여넣기
- 해시태그: 캡션 마지막 또는 첫 댓글에 추가

---

## 주간 루틴 (화·목 각 5분)

| 작업 | 소요 시간 |
|:-----|:---------|
| Claude에 캐러셀 요청 → CSV 저장 | 3분 |
| Canva Bulk Create 실행 → 다운로드 | 3분 |
| 인스타 업로드 + 캡션 + 예약 | 4분 |
| **합계** | **~10분/회** |

---

## 자주 발생하는 오류

| 오류 | 원인 | 해결 |
|:-----|:-----|:-----|
| 텍스트가 변수명 그대로 출력됨 | 매핑 안 됨 | Bulk Create에서 컬럼 재매핑 |
| 글자가 박스 밖으로 넘침 | 텍스트 너무 김 | Claude에 "15자 이내로" 재요청 |
| CSV 쉼표 오류 | 따옴표 누락 | Claude 출력 CSV 재확인 |
