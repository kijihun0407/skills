# Directive: 릴스 편집 SOP (캡컷 기준)

> Layer 1. 대본 완성 후 편집 → 업로드 준비까지 전 과정.

## 목표

30~40초 릴스를 캡컷으로 편집해 발행 준비 완료 상태로 만든다.

## 편집 핵심 원칙 3가지

1. **컷 전환 속도** — 2~4초마다 장면 전환 (시청자 집중력 유지)
2. **텍스트 오버레이** — 핵심 키워드를 화면에 띄워 정보 전달력 강화
3. **효과음 활용** — 전환·강조·후킹 포인트에 효과음 배치

## 편집 체크리스트

- [ ] **자막 추가** — 무음 시청자 50%+ 대응 (2026: 자막 없으면 도달 20~30% 감소)
- [ ] **후킹 첫 프레임** — 첫 1초에 텍스트 오버레이 또는 강렬한 비주얼
- [ ] **인기 상승 음악** — 인스타 앱 "인기 상승" 태그 음악 사용
- [ ] **비율 확인** — 9:16 (1080x1920px) 필수
- [ ] **고화질 내보내기** — 1080p 이상
- [ ] **워터마크 제거** — TikTok/CapCut 워터마크 반드시 제거 후 업로드 (알고리즘 억제)

## AI 편집 도구 (편집량 많을 때 보조 활용)

| 도구 | 기능 | 가격 | 추천 상황 |
|:---|:---|:---|:---|
| **Submagic** | AI 자동 자막 + 하이라이트 강조 | $20/월 | 자막 편집 시간 90% 단축 |
| **Gling.ai** | 침묵/"음…어…" 자동 제거 | $12/월 | 편집 시간 80% 단축 |
| **Opus Clip** | 긴 영상 → 숏폼 자동 추출 | 무료/\$19/월 | 유튜브→릴스 변환 |
| **Descript** | 텍스트 수정 = 영상 편집 | 무료/\$24/월 | 반복 콘텐츠 제작자 |

> 핵심: AI 도구는 보조. 캡컷으로 기본 익히고, 편집량 많아지면 추가.

## 촬영 방식별 편집 가이드

| 방식 | 특징 | 편집 포인트 |
|:---|:---|:---|
| **얼굴 노출 (토킹헤드)** | 퍼스널 브랜딩 강력 | 컷 전환 빠르게, 자막 필수 |
| **화면녹화 + 나레이션** | 기술/교육 분야 효과적 | 클릭 확대 효과 활용 |
| **움짤(GIF) 활용** | 얼굴 비노출, 부담 적음 | 임팩트 있는 GIF 선정이 핵심 |
| **AI 아바타** | 얼굴 비노출, 진정성 주의 | 화면녹화와 조합 권장 |
