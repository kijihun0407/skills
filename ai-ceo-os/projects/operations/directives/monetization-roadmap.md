# Directive: 수익화 로드맵 관리 SOP

> Layer 1. 수익화 진행 상황을 추적하고 다음 단계를 결정한다.

## 목표

팔로워 규모에 맞는 수익화 경로를 선택하고, 60일 단위로 진행 상황을 추적한다.

## 수익화 의사결정 트리

### 현재 팔로워 0~1,000명

| 본업 유무 | 추천 경로 | 첫 수익 | 월 목표 |
|:---|:---|:---|:---|
| 직장인/자영업자 | **경로 A**: 콘텐츠 + 커뮤니티 빌딩 | 2~3개월 | 30~100만원 |
| 전업 크리에이터 | **경로 B**: 1:1 컨설팅/에이전시 빠른 런칭 | 1~2개월 | 100~300만원 |

> 경로 A 이번 주 액션: 오픈채팅방 개설 + 릴스 CTA에 입장 유도

### 팔로워 1,000~10,000명

| 수익화 경험 | 추천 경로 | 월 목표 |
|:---|:---|:---|
| 없음 | **경로 C**: 전자책/프롬프트팩으로 첫 매출 | 50~200만원 |
| 있음 | **경로 D**: 상품 사다리 구축 (1만→10만→100만원) | 200~500만원 |

> 경로 C 이번 주 액션: 스토리 수요조사 "이런 자료 만들면 살 사람?"

### 팔로워 10,000명+

| 팀 유무 | 추천 경로 | 월 목표 |
|:---|:---|:---|
| 1인 운영 | **경로 E**: VOD + 디지털 제품 + DM 자동화 | 300~1,000만원 |
| 팀 있음 | **경로 F**: 부트캠프/SaaS/에이전시 사업화 | 1,000만원+ |

## 60일 수익화 체크포인트

### 1~2주차: 기반 구축
- [ ] 커뮤니티 개설 (오픈채팅방)
- [ ] 프로필 링크 셋팅 (리틀리/링크트리)
- [ ] 수익화 모델 선택 (6가지 중 1~2개)

### 3~4주차: 상품 기획 + 수요 검증
- [ ] 판매할 상품/서비스 확정
- [ ] 스토리/커뮤니티에서 수요 검증
- [ ] 판매 플랫폼 셋팅

### 5~6주차: 판매 시작 + 첫 매출
- [ ] 런칭 콘텐츠 제작 (릴스 + DM 자동화 연동)
- [ ] 첫 판매 달성
- [ ] 피드백 수집 → 상품 개선

### 7~8주차: 최적화
- [ ] 전환율 분석 (프로필 링크 클릭 → 구매율)
- [ ] 광고 단가표 준비 + 역제안 발송
- [ ] 다음 60일 계획 수립
