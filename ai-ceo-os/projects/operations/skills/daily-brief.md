# 스킬: 주간 브리핑

> 매주 시작 시 실행 — 지난 주 성과 + 이번 주 계획 산출.

---

## 입력

```
지난 주 완료한 것: [목록]
지난 주 못 한 것: [목록]
이번 주 목표: [목록] (없으면 CLAUDE.md 목표 기준 자동 설정)
```

---

## 출력 형식

```markdown
# 주간 브리핑 — {날짜}

## 지난 주 성과
- 완료: [목록]
- 미완: [목록] + 원인 1줄

## 이번 주 계획
| 요일 | 작업 | 예상 소요 시간 |
|:---|:---|:---:|
| 월 | | |
| 화 | | |
...

## 이번 주 핵심 1가지
> 이번 주 반드시 완료해야 할 단 하나: [항목]

## 병목 경고
- [있으면] 지난 주 반복된 지연 작업 → 스킬화 제안
```

---

## 품질 기준

| ID | 기준 | 합격 조건 |
|:---|:---|:---|
| B1 | 핵심 1가지 | 반드시 1개만 (2개 이상 = 집중 분산) |
| B2 | 소요 시간 | 직장 병행 가능한 현실적 시간 |
| B3 | 병목 식별 | 반복 지연 항목 있으면 반드시 언급 |
