# directives/ — 본사 SOP 지시서

> Layer 1: Directive. AI가 읽고 실행하는 업무 매뉴얼.
> 자연어로 작성된 Markdown. "어떤 목표로, 어떤 입력을 받아, 어떤 도구를 쓰고, 무엇을 출력하는가"를 정의.

## 원칙

- Directive는 살아있는 문서다. AI가 새로운 엣지케이스를 발견하면 이 파일을 업데이트한다.
- CEO 허락 없이 Directive를 삭제·덮어쓰기 금지. 개선은 가능.
- 새 SOP 추가 시 이 README에 항목 추가.
