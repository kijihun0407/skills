# Directive: 릴스 대본 작성 → 발행 준비

> Layer 1 SOP. 주제 선정 완료 후 릴스 대본을 완성본으로 만드는 전 과정.

## 목표

선정된 주제로 30~40초 릴스 대본 완성본 제작 + 발행 준비 패키지 완성.

## 입력값

- 선정된 주제 (content-research.md 결과물)
- 타겟 고객 여정 단계
- 발행 예정일

## 실행 순서 (Orchestration 가이드)

1. `projects/marketing/skills/reels-script.md` 스킬 파일 참조
2. 대본 작성 (품질기준: `.claude/rules/품질기준.md`)
3. 검수 기준 자동 체크
4. 해시태그 5개 이내 선정
5. CTA 1개 확정
6. `projects/marketing/outputs/{연도}/{월}/{일}/[릴스]_제목.md` 저장

## 출력 형식

```markdown
# [릴스] {제목}

**타겟 단계**: {1~6단계}
**발행 예정**: {날짜}
**예상 시청 시간**: {초}

---

## 대본

{대본 전문}

---

## 발행 패키지

- **해시태그**: #{1} #{2} #{3} #{4} #{5}
- **CTA**: {CTA 문구}
- **썸네일 텍스트 제안**: {문구}
```

## 품질 체크리스트

- [ ] 첫 문장 3초 내 후킹 (질문/충격/반전)
- [ ] 전체 길이 30~40초
- [ ] 모든 줄 인사이트/꿀팁 (filler 없음)
- [ ] CTA 정확히 1개
- [ ] 해시태그 5개 이내
