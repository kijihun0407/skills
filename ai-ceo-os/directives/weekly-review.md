# Directive: 주간 성과 리뷰 + 다음 주 계획

> Layer 1 SOP. 매주 성과를 추적하고 다음 주 방향을 조정한다.

## 목표

이번 주 콘텐츠·수익·팔로워 성과를 정리하고, 다음 주 우선순위를 확정한다.

## 입력값

- 이번 주 발행된 콘텐츠 목록
- 팔로워 변화 (있으면)
- 수익 데이터 (있으면)
- `tasks/todo.md` 현재 상태

## 실행 도구

1. `execution/` 폴더에 통계 수집 스크립트가 있으면 먼저 실행
2. 없으면: CEO에게 데이터 요청 후 진행 (Zero-Inference 원칙)

## 출력 형식

```markdown
# 주간 리뷰 — {날짜}

## 이번 주 성과 요약

| 지표 | 목표 | 실제 | 달성률 |
|:---|:---|:---|:---|
| 릴스 발행 수 | 3개 | - | - |
| 팔로워 증가 | - | - | - |
| 전자책 판매 | - | - | - |

## 잘된 점 (WIN)
...

## 개선할 점
...

## 다음 주 우선순위 TOP 3
1. ...
2. ...
3. ...

## directives 업데이트 필요 사항
...
```
