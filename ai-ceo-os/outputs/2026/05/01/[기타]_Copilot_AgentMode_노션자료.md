## 🔍 Copilot Agent Mode란?

기존 Copilot은 **'제안'** 만 했습니다.
Agent Mode는 **'실행'** 합니다.

- 명령 한 번 → AI가 계획 수립 → 단계별 자동 실행
- 작업 진행 상황을 사이드바에서 실시간 확인 가능
- 중간에 멈추거나 되돌리는 것도 가능
- ChatGPT/Claude 창 따로 열 필요 없음 — 오피스 안에서 바로 실행

**Microsoft 공식 데이터**: 생산성 30~60% 향상 / 주간 활성 사용자 2,000만 명

## ✅ 활성화 방법 (3단계)

> 조건: Microsoft 365 구독 (Personal / Family / Business 모두 가능)

1. Word / Excel / PowerPoint 중 하나 실행
2. 우측 상단 **Copilot 아이콘** 클릭 → 사이드바 열기
3. 사이드바 상단에서 **'Agent Mode'** 탭 선택 → 명령 입력 후 실행

## 📊 Excel 실전 활용 — 데이터 분석 자동화

**효과**: 데이터 분석 시간 **35% 단축** (Microsoft 데이터)

**바로 쓰는 프롬프트**

- "이 데이터로 부서별 매출 피벗 테이블 만들고 차트로 시각화해줘"
- "C열과 D열을 비교해서 차이가 10% 이상인 항목만 빨간색으로 표시해줘"
- "이 데이터에서 이상치(outlier)를 찾아서 별도 시트로 분리해줘"
- "월별 추이를 꺾은선 그래프로 만들고 목표치 라인도 추가해줘"

**꿀팁**

- 수식 모르는 함수도 '이렇게 계산해줘'로 설명하면 AI가 자동 완성
- 데이터 범위를 먼저 선택하고 명령하면 정확도 높아짐
- AI가 만든 수식은 반드시 결과값 1~2개 수동 검증 권장

## 📝 Word 실전 활용 — 문서 작성 자동화

**효과**: 보고서 작성 시간 **55% 단축** (Microsoft 데이터)

**바로 쓰는 프롬프트**

- "이 보고서를 임원 보고용으로 3페이지 이내로 요약해줘"
- "지난달 보고서 형식 유지하면서 이번 달 내용으로 다시 써줘"
- "이 회의록을 액션 아이템 중심으로 정리해줘, 담당자와 기한 포함해서"
- "이 문서의 어투를 공식적인 비즈니스 문체로 바꿔줘"

**꿀팁**

- 참고할 이전 문서를 함께 열어두면 형식·어투를 그대로 유지해줌
- "~하되 ~는 빼줘" 조건을 구체적으로 줄수록 원하는 결과에 가까워짐
- 초안 완성 후 "더 간결하게 줄여줘" 2차 명령으로 다듬기 가능

## 📑 PowerPoint 실전 활용 — 슬라이드 자동화

**효과**: 발표 자료 제작 시간 **45% 단축** (Microsoft 데이터)

**바로 쓰는 프롬프트**

- "이 PPT에 최신 수치 반영하고 디자인은 그대로 유지해줘"
- "각 슬라이드에 발표자 노트를 2~3문장씩 추가해줘"
- "이 내용을 5장짜리 경영진 보고용 슬라이드로 만들어줘"
- "슬라이드 전체 폰트를 통일하고 색상 테마를 파란색 계열로 바꿔줘"

**꿀팁**

- 템플릿이 있으면 먼저 열어두고 "이 형식에 맞춰서 만들어줘"로 시작
- 슬라이드 순서 변경도 명령 하나로 가능 ("3번 슬라이드를 1번으로 이동해줘")
- 이미지 삽입은 아직 수동 필요 — 텍스트/레이아웃 자동화에 집중
