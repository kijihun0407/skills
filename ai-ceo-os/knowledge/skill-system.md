# 스킬 시스템 — 자동화 3단계 + 역설계법

> "반복하면 시스템으로" — 같은 일을 3번 수동으로 하면, 그때 시스템으로 만든다.

---

## 자동화 3단계

| 단계 | 이름 | 누가 시작하나 | 비유 | 위치 |
|:---|:---|:---|:---|:---|
| **1단계** | **스킬 (skills/)** | 내가 시킴 — "릴스 대본 써줘" | 업무 매뉴얼 | `.claude/skills/` |
| **2단계** | **커맨드 (commands/)** | 내가 시킴 — "/광고분석" 한 줄 | 원클릭 보고서 | `.claude/commands/` |
| **3단계** | **크론 (cron)** | 아무도 안 시킴 — 매일 자동 실행 | 24시간 자동 비서 | `crontab` |

### 핵심 차이
- **스킬**: "대본 써줘" → AI가 스킬 파일 참조해서 실행 → 내가 시작, AI가 기준대로 실행
- **커맨드**: "/광고분석" → 스킬보다 복잡한 워크플로우를 1줄로 → 내가 시작, 여러 단계 자동 연결
- **크론**: 새벽 5시 자동 실행 → 아침에 디스코드로 결과 도착 → 아무도 안 시킴, 매일 알아서

## 스킬 역설계 3단계 (실전 제작법)

> "스킬은 설계하는 게 아니라, 대화하다가 발견하는 겁니다."

### Phase 1: 대화 (수동)
1. 평소처럼 AI에게 일을 시킨다
2. 결과물이 70점 → 구체적 피드백으로 90점으로 올린다
3. 레퍼런스나 데이터가 있으면 같이 제공

```
"릴스 대본 써줘" → 피드백 → "후킹이 약해, 질문형으로 바꿔줘" → 반복
```

### Phase 2: 스킬화 (역설계)
만족스러운 결과물이 나오면:
```
"지금 이 대화에서 내가 준 지시사항, 피드백, 레퍼런스, 품질 기준을 정리해서
.claude/skills/reels-script.md 파일로 만들어줘.
다음에 '릴스 대본 써줘'라고만 하면 이 기준대로 바로 나오게."
```

→ AI가 내 작업 패턴을 역설계해서 스킬 파일 자동 생성

### Phase 3: 커맨드화 (묶음)
여러 스킬을 매번 순서대로 시킬 때:
```
"이 4단계 작업을 /weekly-content 커맨드 1개로 묶어줘."
```

## RAW → YOURS: AI 결과물을 "내 콘텐츠"로

| 단계 | 설명 | 예시 |
|:---|:---|:---|
| **RAW** | AI가 처음 뱉은 결과물. 무난하지만 개성 없음 | "결론적으로, 이 서비스의 장점은 다양합니다." |
| **YOURS** | 내 경험·내 말투·내 데이터가 들어간 결과물 | "솔직히 저도 처음엔 반신반의했는데, 써보니까 미쳤습니다" |

### AI 냄새 감지법
이런 표현이 보이면 YOURS가 안 된 것:
- "결론적으로"
- "다양한"
- "~라고 할 수 있습니다"
- "종합적으로"

**해결**: `ceo-persona.md`에 내 말투 기준을 넣으면 RAW 단계부터 YOURS에 가까운 결과물이 나온다.

## "잘 만들어진 스킬" 신호

| 잘 만들어진 스킬 ✅ | 다시 다듬어야 할 스킬 ⚠️ |
|:---|:---|
| 주제만 바꿔도 비슷한 품질이 나온다 | 주제마다 품질이 들쑥날쑥하다 |
| 결과물에 추가 수정이 거의 없다 | 매번 같은 부분을 수정하고 있다 |
| "AI 냄새"가 나지 않는다 | "결론적으로", "다양한" 같은 표현이 나온다 |
| 1줄 명령으로 끝난다 | 매번 추가 설명을 덧붙이고 있다 |

→ 오른쪽 증상이 보이면: "매번 수정하는 그 부분을 스킬 품질 기준에 추가해줘"

## AI OS 상태 점검표 (7개)

| # | 점검 항목 | 기준 |
|:---|:---|:---|
| 1 | CLAUDE.md에 내 사업 정보 | 미션/임원/금지행동 포함 |
| 2 | ceo-persona.md에 내 말투 | 좋은 예시 3개 이상 |
| 3 | channel-rules.md에 채널 규칙 | 발행 주기/해시태그/CTA |
| 4 | 에이전트 1명 이상 호출 경험 | 실제로 소집해서 작업 |
| 5 | 스킬 1개 직접 만들고 실행 | 역설계로 제작 |
| 6 | knowledge/에 데이터 1개 이상 입력 | 인사이트 5개 |
| 7 | AI 답변이 내 사업에 맞다고 느낀다 | Before/After 차이 체감 |

**7개 중 5개 이상 ✅ = AI OS 가동 상태**

## 실패 패턴 블랙리스트

1. 에이전트에게 모든 걸 한꺼번에 시킴 → 1에이전트 = 1역할
2. 스킬에 "잘 해줘"만 적음 → 숫자 기준 필수
3. AI 결과를 그대로 사용 → YOURS 단계까지 가야
4. 데이터 안 넣고 분석 시킴 → knowledge/에 진짜 데이터를
5. 실수를 기록 안 함 → lessons.md가 비면 같은 실수 영원히 반복
