## 초기 단계 콘텐츠 전략

초기(팔로워 0~1,000명)에는 데이터가 없으므로 아래 기준으로 운영:

### 콘텐츠 목표: 도달 확장 (고객 여정 1단계)
- 저장율 높은 포맷 우선: **교육형 릴스 70%**
- 바이럴 가능성 높은 후킹: 수치형 + 충격 사실형
- CTA: 저장 유도 (초기에 저장율이 팔로워 성장 지표)

### 벤치마크 기준 (내 데이터 대신 사용)
| 지표 | @biggie_ai 참고 기준 |
|:---|:---|
| 좋은 릴스 저장율 | 5% 이상 |
| 좋은 릴스 도달 | 팔로워 × 3 이상 |
| 교육형 콘텐츠 저장율 | 8%+ (가장 높음) |
| 최적 발행 시간 | 저녁 7~9시 (일반적) |

## 콘텐츠 우선순위 (초기 전략)

```
1순위: 교육형 릴스 (AI 활용법 실전)
  → 저장율 높음, 팔로워 아닌 비팔로워에게 도달
  → 후킹: 수치형 ("이 프롬프트 하나로 2시간 단축")

2순위: 트렌드 속보형 릴스
  → 빠른 공유 유발
  → 후킹: 긴급형 ("오늘 공개됐습니다")

3순위: 카드뉴스
  → 저장 유도 강함
  → 팔로워 전환에 효과적
```

## 2026 인스타그램 알고리즘 변화 (2026-04-19 리서치 반영)

> 출처: polarad.co.kr / studiopupcy.com / openads.co.kr

### 핵심 변화 3가지

**1. 콘텐츠 길이 최적 구간 이동**
- 5~7초 초단 릴스 → 피로 구간 진입, 알고리즘 우대 약화
- **20~40초가 현재 최적 구간** (문제 제시 → 판단 기준 → 정리)
- 기존 전략 그대로 유지 가능 (30~40초 대본 기준과 일치)

**2. 알고리즘 우대 지표 변화**
- 좋아요보다 **저장 + 검색 유입 + 체류 시간** 우대
- "저장율 = 팔로워 성장 지표" 기존 전략 유효성 재확인
- 검색 유입 최적화: 캡션에 키워드 자연 삽입 필요

**3. 진정성 + AI 조합이 승리 공식**
- AI 도구로 제작하되, 크리에이터 목소리(톤앤매너) 유지가 핵심
- 화려한 편집 < 메시지 진정성
- CapCut / VN 기본 편집 + AI 자막 자동화가 효율 최적

### 대본 작성 시 반영 사항
- 후킹 후 문제 상황 → 판단 기준 → 정리 구조 유지
- 캡션 키워드: 검색 유입 고려해 핵심 키워드 앞배치
- 자막 강조: 핵심 키워드 1~2개 폰트 강조 (편집 지시 추가)
