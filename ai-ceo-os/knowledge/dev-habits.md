### 특히 중요한 4가지 (#1/#4/#7/#8)

**#1 기획서 먼저**
PLAN.md 없이 Claude Code 안 켠다. 없으면 AI가 제네릭 쓰레기를 뱉는다.

**#4 환경 확인 먼저 — ★ 40분 장애 사례**
실제 사고 2026-03-24: ainowclass.com 배포하면서 "실제로 결제가 일어나는가?" 확인 안 하고 배포.
실제 결제는 외부 플랫폼에서. 결과: 40분 장애 + 긴급 롤백.

**#7 테스트 안 했으면 완료 아님 — ★ HTTP 200 사고**
실제 사고 2026-04-14: HTTP 200으로 "배포 완료" 판정. 실제는 빈 fallback 페이지.
3중 체크 필수: 상태 코드 + Content-Length + 본문 고유 문자열

**#8 선택지 2개 비교**
"AI야 A와 B 중 뭐가 나아?" 한 번만 물어도 품질 30% 상승.
