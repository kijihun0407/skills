# 플러그인 + MCP — Claude Code 확장 생태계

> 플러그인 = AI에게 새로운 기술을 설치하는 것 (예: 코드 리뷰, 스킬 제작)
> MCP = AI를 외부 서비스에 연결하는 것 (예: 노션, 캘린더, Gmail)

---

## 플러그인 TOP 10 (실사용 우선순위)

| # | 플러그인 | 하는 일 | 호출 방법 | 난이도 | 빈도 |
|:---|:---|:---|:---|:---|:---|
| 1 | **superpowers** | AI에게 "전문 기술 세트"를 자동 부여 (스킬 시스템의 뇌) | 자동 발동 | ⭐ | 매일 |

| 10 | **context7** | 라이브러리 최신 공식 문서를 실시간 참조 (환각 방지) | 자동 발동 (라이브러리 언급 시) | ⭐⭐ | 주 3회 |

### 왕초보 우선 설치 3개
**superpowers + commit-commands + context7** — 이 3개만 있어도 체감이 2배 달라진다.

## 플러그인 활용법

### superpowers — "자동 발동되는 전문가"
- 설치만 해두면 AI가 "릴스 대본 스킬이 필요하네" 하고 스스로 판단해서 전문 모드 전환
- 스킬 34개가 이 플러그인 덕분에 작동

### commit-commands — "Git 명령어 외우지 마세요"
- `/commit` → 변경 파일 확인 + 커밋 메시지 자동 작성 + 커밋
- `/commit-push-pr` → 커밋 + 푸시 + PR 자동 생성

### skill-creator — "역설계로 스킬 찍어내기"
- "나 매주 반복하는 이 작업을 스킬로 만들어줘"
- AI가 인터뷰 질문 5개로 요구사항 파악 → 스킬 파일 자동 생성

### claude-md-management — "AI 메모리 건강검진"
- `/revise-claude-md` → CLAUDE.md 중복/만료 규칙 찾아서 압축 + 최신 규칙 추가 제안
- 월 2회 정기 검진

### playwright — "AI가 직접 브라우저 조작"
- "경쟁사 계정 최근 릴스 3개 스크린샷 뽑아줘"
- "우리 랜딩페이지 모바일에서 어떻게 보이는지 확인해줘"

### context7 — "AI 환각 방지"
- AI는 최신 라이브러리 기준이 아닌 옛날 정보로 답할 때가 있음
- context7 설치 시 최신 공식 문서 자동 조회 후 코드 작성 → 첫 실행에 성공

| 5 | Google Sheets | 구글 시트 | 데이터 읽기/쓰기, 대시보드 |
| 6 | Google Analytics | GA4 | 트래픽/전환 데이터 조회 |
| 7 | YouTube | 유튜브 | 채널 통계, 영상 정보 |
| 8 | Supabase | DB | 데이터베이스 조회/수정 |
| 9 | n8n | 자동화 워크플로우 | 복잡한 자동화 트리거 |

| 구분 | 플러그인 | MCP |
