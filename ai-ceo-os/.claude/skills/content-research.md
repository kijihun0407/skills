# 콘텐츠 리서치 스킬 — AI 트렌드 분석 + 주제 선정

매일 X(트위터), 인스타그램, 유튜브, 스레드에서 AI 최신 트렌드와 이슈를 분석해
이번 주 릴스/카드뉴스 주제 TOP 3를 선정한다.

---

## 실행 조건

- **실행 주기**: 매일 1회 (또는 "오늘 주제 뭐야?" 요청 시)
- **리서치 범위**: 최근 24~48시간 이내 콘텐츠 기준
- **출처 우선순위**: X > 인스타그램 > 유튜브 > 스레드

---

## 실행 절차

### Phase 1 — 트렌드 수집 (WebSearch 사용)

아래 키워드로 최신 트렌드 검색:
```
검색어 세트:
- "AI 트렌드 2026" site:twitter.com OR site:x.com
- "ChatGPT 업데이트" 최신
- "Claude 업데이트" 최신
- "AI 활용법" 인스타그램 트렌드
- "Gemini OR GPT OR Claude OR Sora OR AI" 오늘 이슈
```

수집 기준:
- 공유·좋아요·댓글 수가 높은 것 우선
- 국내 반응 있는 것 우선 (한국어 콘텐츠 또는 한국 반응)
- 실용적으로 써먹을 수 있는 내용 우선

### Phase 2 — 레퍼런스 계정 동향 확인

벤치마크 3인방이 최근 다룬 주제 확인:
- @biggie_ai: 최근 릴스 주제
- @ai_dori_: 최근 릴스 주제
- @ai_ing_: 최근 릴스 주제

→ 이미 다룬 주제는 **앵글 변경** 또는 **제외**

### Phase 3 — 주제 선정 기준 필터링

아래 기준으로 수집된 트렌드 평가:

| 기준 | 체크 | 비중 |
|:---|:---|:---|
| 실용성 — 당장 써먹을 수 있는가? | ✅/❌ | 40% |
| 신선도 — 레퍼런스 계정이 안 다뤘는가? | ✅/❌ | 30% |
| 관심도 — X/인스타 반응이 있는가? | ✅/❌ | 20% |
| 설명 가능성 — 30초 안에 설명 가능한가? | ✅/❌ | 10% |

### Phase 4 — TOP 3 주제 선정 + 앵글 제안

각 주제마다 아래 포맷으로 출력:

```
## 주제 N: [주제명]
- 트렌드 출처: [어디서 뜨고 있는지]
- 추천 앵글: [우리만의 차별화 방향]
- 추천 후킹 방향: 수치형 / 반전형 / 긴급형 중
- 여정 단계: 1~6단계 중
- 예상 저장율: 높음 / 보통
- 레퍼런스 중복: 있음(앵글 변경) / 없음
```
