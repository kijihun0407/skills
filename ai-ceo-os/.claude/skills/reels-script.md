# 릴스 대본 작성 스킬

AI 크리에이터 채널용 인스타그램 릴스 대본을 완성본으로 즉시 산출한다.
주제만 주면 3버전 후킹 + 최종 대본 1편 + 캡션 + 검수 결과까지 한 번에 납품.

---

## 페르소나 기준 (자동 적용)

- **말투 기본**: @biggie_ai 스타일 — 단정형, 군더더기 없음
- **설명 구간**: @ai.sangkyun 스타일 — "쉽게 말하면..." 대화체
- **절대 금지**: 인사말, "오늘은~", "결론적으로", "다양한"
- **채널 현황**: 초기 단계 → KPI 기본값 = 도달확장 (저장 유도)

---

## 입력 형식

```
주제: [AI 관련 주제 1줄]
여정 단계: 1~6 (미입력 시 기본값 1단계)
KPI: 도달확장 / 리드수집 / 구매전환 (미입력 시 도달확장)
```

---

## 실행 절차

### Phase 0 — 주제 산출 (content-research 스킬 실행)

주제가 입력되지 않은 경우 자동으로 content-research 스킬을 실행한다.

```
@.claude/skills/content-research.md 실행
→ TOP 3 주제 산출
→ 1순위 주제를 이 스킬의 [주제] 입력값으로 사용
```

주제가 이미 입력된 경우 Phase 0 건너뜀.

### Phase 1 — 리서치 (내부)
- 레퍼런스 3개 (@biggie_ai / @ai_dori_ / @ai_ing_) 유사 콘텐츠 앵글 확인
- 차별화 포인트 1개 결정 → 대본에 반영

### Phase 2 — 후킹 3버전 작성
| 버전 | 유형 | 패턴 |
|:---:|:---|:---|
| A | 수치형 | "N명 중 N명이 모르는 ~" / "N초 만에 ~" |
| B | 반전형 | "사실 ~이 아닙니다" / "AI를 잘 쓰는 사람은 ~을 안 합니다" |
| C | 긴급형 | "오늘 공개됐습니다" / "지금 안 쓰면 뒤처집니다" |

→ 3버전 제시 후 **CCO 관점에서 최적 버전 선택** (CMO 전환율 검수 병행)

### Phase 3 — 완성 대본 작성
선택된 후킹으로 30~40초 분량 완성:
```
[후킹 — 첫 3초]   ← biggie_ai 스타일 (단정형, 수치/사실)
[핵심 메시지]      ← 인사이트 전달 (매 줄이 꿀팁)
[비유 설명]        ← "쉽게 말하면..." sangkyun 스타일
[CTA — 마지막 5초] ← 명사 키워드 1개만
```

### Phase 4 — 검수 + 납품
품질 기준 자동 체크 후 결과 첨부

---

## 출력 형식

```markdown
## 후킹 3버전
- A (수치형): [문장]
- B (반전형): [문장]
- C (긴급형): [문장]
→ 선택: [버전] / 이유: [한 줄]

---

## 최종 대본
[대본 전문]

---

## 캡션
[150자 이내]
#해시태그1 #해시태그2 #해시태그3 #해시태그4 #해시태그5

---

## 검수 결과
| ID | 기준 | 결과 |
|:---|:---|:---|
| Q1 | 첫 문장 후킹 | ✅/❌ |
| Q2 | 글자 수 (180~250자) | ✅/❌ [N자] |
| Q3 | CTA 1개 | ✅/❌ |
| Q4 | 인사말 없음 | ✅/❌ |
| Q5 | "쉽게 말하면..." 포함 | ✅/❌ |
| Q6 | 구체적 숫자/사례 | ✅/❌ |

📋 검수: 전항목 합격 / [N]건 플래그
```
