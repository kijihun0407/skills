# /daily-content — 오늘 콘텐츠 원스톱 실행

> 리서치 → 주제 선정 → 릴스 대본까지 한 번에.
> 매일 아침 또는 콘텐츠 제작 시작 전 실행.

---

## 실행 순서

### STEP 1 — 트렌드 리서치 + 주제 TOP 3 선정

@.claude/skills/content-research.md 실행

출력:
- 오늘의 주요 트렌드 요약 3줄
- 추천 주제 TOP 3 (1순위 즉시 제작 추천)
- 레퍼런스 계정 중복 주의 목록
- 이번 주 주제 캘린더 제안

---

### STEP 2 — 1순위 주제로 릴스 대본 즉시 제작

STEP 1에서 선정된 1순위 주제를 자동으로 입력값으로 사용.

@.claude/skills/reels-script.md 실행
- 주제: [STEP 1 — 1순위 주제]
- 여정 단계: 1 (기본값 — 도달 확장)
- KPI: 도달확장 (기본값)

출력:
- 후킹 3버전 (수치형/반전형/긴급형) + 최적 버전 선택
- 최종 대본 완성본
- 캡션 + 해시태그
- 검수 결과

---

### STEP 3 — 산출물 저장

완성된 대본을 아래 경로에 저장:
```
ai-ceo-os/projects/marketing/outputs/{YYYY}/{MM}/{DD}/[릴스]_{주제요약}.md
```

---

## 사용법

```
/daily-content
```

또는

```
오늘 콘텐츠 만들어줘
```

---

## 옵션 (선택 입력)

```
/daily-content 주제: [직접 입력한 주제]
```
→ STEP 1 건너뛰고 STEP 2부터 즉시 실행.

```
/daily-content 여정: 3 KPI: 리드수집
```
→ 고객 여정 3단계, 리드수집 목적 대본 제작.
